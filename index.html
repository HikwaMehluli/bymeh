Mobile Responsive

This starter theme is fully mobile responsive, which means it will scale perfectly on mobile, tablet and computer screens with ease.

Sliders

The slider included is Owl Carousel 2, a touch enabled jQuery plugin that lets you create beautiful responsive carousel sliders.

Grid Layout

The grid is a mobile responsive 4-column flexbox based grid with a max width of 1140px, with a simple syntax which makes coding responsive layouts easy. Try resize the browser.
